## Implementation

### New file: `src/minidic/hotkey.py`
- `GlobalHotkeyListener` class using macOS `CGEventTap` (listen-only mode)
- Runs `CFRunLoop` on a daemon thread to receive keyboard events
- Filters for F5 key-down (virtual keycode 96)
- Ignores auto-repeat events and debounces rapid presses (300ms)
- Requires Accessibility permission; prints clear error if not granted

### Modified: `src/minidic/main.py`
- Added `--hotkey` CLI flag
- Added `run_hotkey()` function implementing the F5 daemon mode
- State machine: idle → recording → transcribing → idle
  - F5 in idle: spawns recording thread
  - F5 in recording: signals stop via `threading.Event`
  - F5 in transcribing: ignored
- Respects `--max-duration`, `--clipboard`, `--device`, `--model` flags
- Original Enter/Ctrl-C mode unchanged (default without `--hotkey`)

### Usage
```
minidic --hotkey              # F5 global hotkey mode
minidic --hotkey --clipboard  # Use clipboard injection
minidic --hotkey -v           # With debug logging
```
