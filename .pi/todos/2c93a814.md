Implemented `src/minidic/inject.py`:

- `inject_text(text, per_char=False)` — primary method using CGEventKeyboardSetUnicodeString with 20-char chunking
- Key down + key up pairs per chunk via `CGEventPost(kCGHIDEventTap, ...)`
- Reusable `CGEventSourceCreate(kCGEventSourceStateHIDSystemState)` source
- `per_char=True` mode for problematic apps (1 char at a time)
- 1ms inter-chunk delay for reliability
- `inject_via_clipboard(text)` — fallback using pbcopy + simulated Cmd+V
- Graceful error if Accessibility permission not granted
